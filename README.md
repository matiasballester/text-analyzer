== Building

To launch your tests:
```
./mvnw clean test
```

To package your application:
```
./mvnw clean package
```

To run your application:
```
./mvnw clean compile exec:java
```

Please first run docker compose up to start postgres docker image. It's required by the tool.
```
docker compose up
```
